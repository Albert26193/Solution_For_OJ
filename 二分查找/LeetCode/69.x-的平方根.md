# 69.x-的平方根

> date: 2020-03-24
> tag : greedy

---
## 题目描述：
实现 int sqrt(int x) 函数。<br>
计算并返回 x 的平方根，其中 x 是非负整数。<bf>
由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。<br>

示例:
```
输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。
```

---
## 题目解答：
首先想到的就是二分查找，这是没有什么疑问的，问题在于一些细节的处理上，第1，优先使用除法而不是乘法，因为mid*mid的数值可能超过int范围的上限，而除法不存在这个问题。第2，除法的使用需要注意排除除数为0的情况。
