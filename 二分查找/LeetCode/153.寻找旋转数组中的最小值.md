# 寻找旋转数组中的最小值

>date: 2020-03-24
>tag : greedy

---
## 题目描述：
假设按照升序排序的数组在预先未知的某个点上进行了旋转。例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] 。

请找出其中最小的元素。

示例：
```
输入：nums = [3,4,5,1,2]
输出：1
```

---
## 题目解答：
问题的关键在于寻找那个转折的点。思路是通过二分查找进行寻找。<br>
最初时刻，left = 0 && right = len - 1<br>
如果nums[mid] < nums[left]，说明此时的mid点已经在旋转数组后半部分上了。<br>
如果nums[mid] > nums[right]，说明此时的mid点仍然在旋转数组前半部分上。<br>
