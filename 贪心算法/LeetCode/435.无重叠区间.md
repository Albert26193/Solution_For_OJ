# 435.无重叠区间

> date: 2020-03-22<br>
> tag : greddy

---
## 题目描述：
给定一个区间的集合，找到需要移除区间的最小数量，使剩余区间互不重叠。

注意:

可以认为区间的终点总是大于它的起点。
区间 [1,2] 和 [2,3] 的边界相互“接触”，但没有相互重叠。

示例 1:
```
输入: [ [1,2], [2,3], [3,4], [1,3] ]

输出: 1

解释: 移除 [1,3] 后，剩下的区间没有重叠。
```

---
## 题目解答：
题目可以换一个角度来理解：<br>
如何保留尽可能多的不重叠区间？
一定存在某种解法是最优解法，我们设法对其进行改良，进而总结出一些特征。
该序列中的某一个区间为[x,y],它的上一个区间右端点必然小于x，不妨设其为z，对于所有大于z的区间，我们一定选择其右边界最小的那一个。因为选择右边界非最小的那一个，其必不优于选右边界最小的。

具体实现上，我们选择右区间最小的那一个，将其右边界设为limit，所有左边界大于limit的区间，选择右边界最小的那一个成为第2个非重叠区间。以此类推，直到所有区间都被遍历。
